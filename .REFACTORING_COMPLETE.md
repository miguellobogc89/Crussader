## 🎯 Flujo Final (Post-FASE-3)

```
┌─────────────────────────────────────────────────────────────┐
│                    FLUJO VALIDADO                           │
├─────────────────────────────────────────────────────────────┤

1. CONECTAR GOOGLE
   GET  /connect                  → Redirige a Google OAuth
   GET  /callback?code=...        → Crea ExternalConnection + calendarios

2. ACTUALIZAR CALENDARIOS
   POST /sync-calendars           → Refresh lista de Google

3. IMPORTAR GOOGLE → CRUSSADER
   POST /sync                     → Eventos desde google_context calendars
                                    Rango: 90 días

4. EXPORTAR CRUSSADER → GOOGLE
   POST /backfill                 → Citas Crussader → crussader_mirror calendar
                                    Rango: 3 meses

5. DESCONECTAR GOOGLE
   DELETE /disconnect             → Marca calendarios inactivos
                                    Cancela citas importadas
                                    MANTIENE citas internas

QUERY ESTADO:
   GET  /connected                → Lista calendarios conectados
   GET  /status                   → Estado de conexión

├─────────────────────────────────────────────────────────────┤
│  Total Endpoints: 8 (limpio, enfocado, sin deuda técnica)   │
└─────────────────────────────────────────────────────────────┘
```
