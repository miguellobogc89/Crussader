## Endpoints Restantes (Núcleo) ✅

```
GET  /api/integrations/google/calendar/connect
     → Inicia OAuth flow

GET  /api/integrations/google/calendar/callback
     → Recibe OAuth code, crea ExternalConnection

POST /api/integrations/google/calendar/sync-calendars
     → Actualiza lista de calendarios desde Google

POST /api/integrations/google/calendar/sync
     → Importa eventos de google_context calendars → Crussader

POST /api/integrations/google/calendar/backfill
     → Exporta appointments de Crussader → Google (crussader_mirror)

DELETE /api/integrations/google/calendar/disconnect
       → Desconecta, marca calendarios inactivos

GET  /api/integrations/google/calendar/connected
     → Lista calendarios conectados
     
GET  /api/integrations/google/calendar/status
     → Estado de conexión
```

**Total**: 8 endpoints (limpio, enfocado, sin deuda técnica)
