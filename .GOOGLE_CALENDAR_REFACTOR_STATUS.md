## 📋 FLUJO ESPERADO - VALIDADO

**Después de FASE 1**, el flujo es:**

### 1️⃣ **Conectar Google**
```
GET  /api/integrations/google/calendar/connect
     → Redirige a Google OAuth
     
GET  /api/integrations/google/calendar/callback?code=...
     → Crea ExternalConnection + calendarios (crussader_mirror + google_context)
```

### 2️⃣ **Sincronizar Calendarios** (Refresh lista)
```
POST /api/integrations/google/calendar/sync-calendars
     → Actualiza external_calendar list desde Google
```

### 3️⃣ **Importar Google → Crussader** (Read Google)
```
POST /api/integrations/google/calendar/sync
     {
       "companyId": "...",
       "locationId": "..."
     }
     → Importa eventos de google_context calendars
     → Crea Appointments con externalProvider="google-calendar"
     → Rango: últimos 90 días
```

### 4️⃣ **Exportar Crussader → Google** (Write Google)
```
POST /api/integrations/google/calendar/backfill
     {
       "locationId": "..."
     }
     → Exporta Appointments sin externalProvider a crussader_mirror
     → Crea eventos en Google, actualiza Appointment.externalEventId
     → Rango: últimos 3 meses
```

### 5️⃣ **Desconectar Google**
```
DELETE /api/integrations/google/calendar/disconnect
       → Marca google_context como inactive
       → Cancela appointments importados
       → Mantiene citas internas de Crussader
```
