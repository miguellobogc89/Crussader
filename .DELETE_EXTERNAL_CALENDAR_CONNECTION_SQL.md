## 🛡️ SQL SEGURO - Opción A: Remplazar relación antes de eliminar

### Paso 1: Cambiar external_calendar_event.connection_id a external_calendar.id

```sql
-- Primero, verificar que external_calendar tiene connection_id
SELECT COUNT(*) FROM external_calendar LIMIT 1;

-- Crear nueva columna en external_calendar_event
ALTER TABLE external_calendar_event 
ADD COLUMN external_calendar_id_new UUID;

-- Copiar datos (usar external_calendar.id para cada external_calendar_event)
UPDATE external_calendar_event ece
SET external_calendar_id_new = ec.id
FROM external_calendar ec
WHERE ec.external_calendar_id = ece.external_calendar_id
  AND ec.provider = ece.provider;

-- Verificar que se llenaron
SELECT COUNT(*) FROM external_calendar_event 
WHERE external_calendar_id_new IS NOT NULL;

-- Eliminar foreign key antigua
ALTER TABLE external_calendar_event 
DROP CONSTRAINT external_calendar_event_connection_id_fkey;

-- Eliminar columna connection_id
ALTER TABLE external_calendar_event 
DROP COLUMN connection_id;

-- Renombrar nueva columna
ALTER TABLE external_calendar_event 
RENAME COLUMN external_calendar_id_new TO external_calendar_id;

-- Crear nueva foreign key
ALTER TABLE external_calendar_event
ADD CONSTRAINT external_calendar_event_external_calendar_id_fkey
FOREIGN KEY (external_calendar_id) 
REFERENCES external_calendar(id) ON DELETE CASCADE;
```

### Paso 2: Eliminar tabla external_calendar_connection

```sql
DROP TABLE IF EXISTS external_calendar_connection CASCADE;
```

### Paso 3: Actualizar Prisma schema

```prisma
// EN: prisma/schema.prisma

// ANTES:
model external_calendar_event {
  id                           String                       @id
  connection_id                String
  ...
  external_calendar_connection external_calendar_connection @relation(fields: [connection_id], references: [id], onDelete: Cascade, onUpdate: NoAction)
}

// DESPUÉS:

  id                           String                       @id
  external_calendar_id         String
  ...
  external_calendar            external_calendar @relation(fields: [external_calendar_id], references: [id], onDelete: Cascade, onUpdate: NoAction)
}
```

### Paso 4: Prisma migration

```bash
npx prisma migrate dev --name consolidate_external_calendar_connection
```
